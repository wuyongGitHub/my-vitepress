## v1.0.11（修复热力图数据NaN错误）

```
解决问题：解决数据稳定性问题，调用时防止出现NaN的错误
1. 数据全为 0：你的热力图数据全是 0，导致 min = 0，max = 0，计算高度时出现除以0的情况。
2. 高度计算问题：当 max - min = 0 时，(value - min) / (max - min) 得到 NaN。
```
